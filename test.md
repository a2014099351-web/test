# 标准 / 科学模式手工测试步骤

| 项 | 内容 |
|----|------|
| 范围 | 标准模式 + 科学模式的计算逻辑、状态机、内存、历史、快捷键、模式切换 |
| 不测 | 长度/重量换算、皮肤/图标包细节（仅在切换回算时附带确认不崩溃） |
| 被测程序 | `build\bin\Debug\kcalculator.exe`（或 Release） |
| 记录方式 | 每步勾选 Pass / Fail，Fail 时写「实际现象」 |
| 约定 | `%` / `mod` = **求余**（非百分比）；括号不匹配 **报错不补齐** |

**启动前**：`cmake --build build --config Debug --target kcalculator`，然后运行上述 exe。

| 0.3 | 确认 MC / MR | 内存空时 MC、MR **置灰不可点**；M+ / M- / MS 可用 | ☐ |
| 0.4 | 打开历史侧栏 | 空态提示（如 There's no history yet.） | ☐ |
